# Rerooted 🌱

A recipe curation app that helps you cook smarter with what you already have, 
reducing food waste one meal at a time.

## What it does

You input the ingredients sitting in your fridge and Rerooted suggests recipes 
you can make right now. No more throwing out food because you didn't know what 
to do with it.


## Why I built it

Food waste is a massive sustainability problem. Most of it happens at home — 
ingredients bought with good intentions that never get used. Rerooted tries to 
solve the last-mile problem: you have the food, you just need the idea.

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Bolt Marketplace API**
- Deployed on **Vercel**

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Roadmap

- [ ] **Photo upload** — snap your fridge and let the app identify ingredients 
automatically using computer vision
- [ ] **Expiry-aware sorting** — ingredients closest to going bad get prioritised 
in recipe suggestions first
- [ ] **Pantry tracker** — save your regular ingredients so you don't have to 
re-enter them each time
- [ ] **Nutritional insights** — understand what your leftovers-based meals are 
actually giving you
